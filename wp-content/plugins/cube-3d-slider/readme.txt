Display cube 3D slider in your website.

== Description ==

Display beautiful 3D cube sliders. Here some demonstrations : http://www.info-d-74.com/cube-3d-slider-plugin-wordpress-demos/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Cube 3D slider menu to create your grid
